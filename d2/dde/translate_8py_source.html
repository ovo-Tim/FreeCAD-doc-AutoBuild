46 def translate(context, text, comment=None):
47  r"""Translate the text using the Qt translate function.
48 
49  It wraps around `QtWidgets.QApplication.translate`,
50  which is the same as `QtCore.QCoreApplication.translate`.
51 
52  Parameters
53  ----------
54  context: str
55  In C++ it is typically a class name.
56  But it can also be any other string to categorize the translation,
57  for example, the name of a workbench, tool, or function
58  that is being translated. Usually it will be the name
59  of the workbench.
60 
61  text: str
62  Text that will be translated. It could be a single word,
63  a full sentence, paragraph, or multiple paragraphs with new lines.
64  Usually the last endline character '\n'
65  that finishes the string doesn't need to be included
66  for translation.
67 
68  Returns
69  -------
70  str
71  A unicode string returned by `QtWidgets.QApplication.translate`.
72 
73  Unicode strings
74  ---------------
75  Reference: https://doc.qt.io/qtforpython/PySide2/QtCore
76 
77  In Qt5 the strings are always assumed unicode
78 
79  >>> QtCore.QCoreApplication.translate(context, text, None)
80  >>> QtWidgets.QApplication.translate(context, text, None)
81  """
82  return Qtranslate(context, text, comment)
