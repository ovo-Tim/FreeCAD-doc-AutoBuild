Detailed Description

Python interface for View3DInventorViewer.

The interface does not offer all methods the c++ View3DInventorViewer counterpart has, respectively also not everything the QuarterWidget and the SoQtQuarterAdaptor offers. It only exposes methods with additional functionality in comparison to the View3DInventorPy class. Everything that can be done from there has no interface here.
