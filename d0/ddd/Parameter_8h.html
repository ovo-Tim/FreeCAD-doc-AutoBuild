Detailed Description

The classes defined here are used to interface with the XML-based FreeCAD config files: user.cfg and system.cfg files. It can parse, get, and store the parameters/configurations for the user's preferences. 3rd party Xerces-C++ XML parser is used to parse and write the XML.
