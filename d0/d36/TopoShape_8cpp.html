◆ defaultAngularDeflection()

double defaultAngularDeflection ( double  linearTolerance)
inline

Derive a roughly proportional default angular deflection from a linear tolerance. This is a work-around for unreliable linear tolerance enforcement in OCC, especially on nurbs surfaces. The intention is to provide sane baseline (default) behavior with linear tolerances only, until OCC is fixed. If needed for specific use cases, explicit angular deflection parameters can still be exposed separately.
