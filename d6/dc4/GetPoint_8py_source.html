103  def getPoint(self, whenDone, start=None):
104  """getPoint(whenDone, start=None) ... invoke Snapper and call whenDone when a point is entered or the user cancels the operation.
105  whenDone(point, obj) is called either with a point and the object on which the point lies if the user set the point,
106  or None and None if the user cancelled the operation.
107  start is an optional Vector indicating from where to start Snapper. This is mostly used when editing existing points. Snapper also
108  creates a dotted line indicating from where the original point started from.
109  If start is specified the Snapper UI is closed on the first point the user enters. If start remains None, then Snapper is kept open
110  until the user explicitly closes Snapper. This lets the user enter multiple points in quick succession.
111  """
