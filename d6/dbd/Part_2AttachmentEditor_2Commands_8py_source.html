40  editAttachment(feature = None,
41  take_selection = False,
42  create_transaction = True,
43  callback_OK = None,
44  callback_Cancel = None,
45  callback_Apply = None)
46  feature: object to attach/modify. If None is supplied, the object is taken from
47  selection.
48  take_selection: if True, current selection is filled into first references (but only
49  if object to be attached doesn't have any references already)
50  create_transaction = if False, no undo transaction operations will be done by the
51  dialog (consequently, canceling the dialog will not reset the feature to original
52  state).
53  callback_OK: function to be called upon OK. Invoked after writing values to feature,
54  committing transaction and closing the dialog.
55  callback_Cancel: called after closing the dialog and aborting transaction.
56  callback_Apply: invoked after writing values to feature.'''
