46 class TaskPanelOrthoArray:
47  """TaskPanel code for the OrthoArray command.
48 
49  The names of the widgets are defined in the `.ui` file.
50  This `.ui` file `must` be loaded into an attribute
51  called `self.form` so that it is loaded into the task panel correctly.
52 
53  In this class all widgets are automatically created
54  as `self.form.<widget_name>`.
55 
56  The `.ui` file may use special FreeCAD widgets such as
57  `Gui::InputField` (based on `QLineEdit`) and
58  `Gui::QuantitySpinBox` (based on `QAbstractSpinBox`).
59  See the Doxygen documentation of the corresponding files in `src/Gui/`,
60  for example, `InputField.h` and `QuantitySpinBox.h`.
61 
62  Attributes
63  ----------
64  source_command: gui_base.GuiCommandBase
65  This attribute holds a reference to the calling class
66  of this task panel.
67  This parent class, which is derived from `gui_base.GuiCommandBase`,
68  is responsible for calling this task panel, for installing
69  certain callbacks, and for removing them.

71  It also delays the execution of the internal creation commands
