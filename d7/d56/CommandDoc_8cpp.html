QString createDefaultExportBasename ( )

Create a default filename from a user-specified format string

Format options are: F - the basename of the .FCStd file (or the label, if it is not saved yet) Lx - the label of the selected object(s), separated by character 'x' Px - the label of the selected object(s) and their first parent, separated by character 'x' U - the date and time, in UTC, ISO 8601 D - the date and time, in local timezone, ISO 8601 Any other characters are treated literally, though if the filename is illegal it will be changed on saving.

The format string is stored in two user preferences (not currently exposed in the GUI): BaseApp/Preferences/General/ExportDefaultFilenameSingle BaseApp/Preferences/General/ExportDefaultFilenameMultiple
