Detailed Description

The MeshGrid allows one to divide a global mesh object into smaller regions of elements (e.g. facets, points or edges) depending on the resolution of the grid. All grid elements in the grid structure have the same size.

Grids can be used within algorithms to avoid to iterate through all elements, so grids can speed up algorithms dramatically.
