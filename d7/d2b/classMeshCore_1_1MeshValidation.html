Detailed Description

The MeshValidation class tries to make a mesh kernel valid with respect to a certain criterion, such as manifoldness, self-intersections, etc. The passed mesh kernel can be modified to fix the errors. The class itself is abstract, hence the method Fixup() must be implemented by subclasses.
