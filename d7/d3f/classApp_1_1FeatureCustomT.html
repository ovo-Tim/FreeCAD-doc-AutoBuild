Detailed Description

template<class FeatureT>
class App::FeatureCustomT< FeatureT >

FeatureCustomT is a template class to be used with DocumentObject or any of its subclasses as template parameter. FeatureCustomT offers a way to add or remove a property at runtime. This class is similar to FeaturePythonT with the difference that it has no support for in Python written feature classes.
