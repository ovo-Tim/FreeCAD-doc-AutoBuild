Detailed Description

template<class Ui>
class Gui::LocationDialogImp< Ui >

This template class is a subclass of LocationDialog using LocationUi and implements the pure virtual methods of its base class. Other dialog-based classes can directly inherit from this class if the location-interface is required. But note, in this case the ui_-header file needs to be included. If this should be avoided the class LocationUi must be used instead of whereas the Ui_-class can be forward declared.
