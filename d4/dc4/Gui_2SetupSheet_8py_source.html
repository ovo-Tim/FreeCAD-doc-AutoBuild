122 class OpTaskPanel:
123  """Editor for an operation's property default values.
124  The implementation is a simplified generic property editor with basically 3 fields
125  - checkbox - if set a default value for the given property is set
126  - name - a non-editable string with the property name
127  - value - the actual editor for the property's default value
128  The specific editor classes for a given property type are implemented in
129  PathSetupSheetOpPrototypeGui which also provides a factory function. The properties
130  are displayed in a table, each field occypying a column and each row representing
