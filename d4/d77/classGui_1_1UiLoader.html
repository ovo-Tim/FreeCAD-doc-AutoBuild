Detailed Description

The UiLoader class provides the abitlity to use the widget factory framework of FreeCAD within the framework provided by Qt. This class extends QUiLoader by the creation of FreeCAD specific widgets.

std::unique_ptr< UiLoader > UiLoader::newInstance ( QObject *  parent = nullptr)
static

Creates a new instance of a UiLoader.

Due to its flaw the QUiLoader upon creation loads every available Qt designer plugin it can find in QApplication::libraryPaths(). Some of those plugins may perform some unexpected actions upon load which may interfere with FreeCAD's functionality. Only way to avoid such behaviour is to reset QApplication::libraryPaths, create a QUiLoader and then restore the libs paths. Hence need for this function to wrap construction.

See also
https://github.com/FreeCAD/FreeCAD/issues/8708
