enum App::Application::PathMatchMode
strong

Path matching mode for getDocumentByPath()

Enumerator
MatchAbsolute 

Match by resolving to absolute file path.

MatchCanonical 

Match by absolute path first. If not found then match by resolving to canonical file path where any intermediate '.' '..' and symlinks are resolved.

MatchCanonicalWarning 

Same as MatchCanonical, but if a document is found by canonical path match, which means the document can be resolved using two different absolute path, a warning is printed and the found document is not returned. This is to allow the caller to intentionally load the same physical file as separate documents.
