DocumentObject * GroupExtension::getGroupOfObject ( const DocumentObject *  obj)
static

Returns the object group of the document which the given object obj is part of. In case this object is not part of a group 0 is returned.

Note
This only returns objects that are normal groups, not any special derived type like GeoFeatureGroups or OriginGroups. To retrieve those please use their appropriate functions
