Detailed Description

template<class Real>
class Wm4::PolynomialSurface< Real >

An implicit surface is defined by F(x,y,z) = 0. This polynomial surface is actually defined as z = f(x,y) = ax^2 + by^2 + cx + dy + exy + g. To use Wm3 routines for implicit surfaces we can write the surface also as F(x,y,z) = f(x,y) - z = 0.
