88 // Magic numbers for Constraint tags
89 // - Positive Tags identify a higher level constraint form which the solver constraint
90 // originates
91 // - Negative Tags represent temporary constraints, used for example in moving operations, these
92 // have a different handling in component splitting, see GCS::initSolution. Lifetime is defined
93 // by the container object via GCS::clearByTag.
94 // - -1 is typically used as tag for these temporary constraints, its parameters are
95 // enforced with
96 // a lower priority than the main system (real sketcher constraints). It gives a nice
97 // effect when dragging the edge of an unconstrained circle, that the center won't move
98 // if the edge can be dragged, and only when/if the edge cannot be dragged, e.g. radius
99 // constraint, the center is moved).
100 enum SpecialTag
101 {
102  DefaultTemporaryConstraint = -1
103 };
