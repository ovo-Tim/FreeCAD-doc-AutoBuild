Detailed Description

A MappedElement combines a MappedName and and IndexedName into a single entity and provides simple comparison operators for the combination (including operator< so that the entity can be sorted, or used in sorted containers).

For sorting purposes, one MappedElement is considered "less" than another if its index compares less (which is first alphabetical, and then by numeric index). If the index of this MappedElement is the same, then the names are compared lexicographically.
