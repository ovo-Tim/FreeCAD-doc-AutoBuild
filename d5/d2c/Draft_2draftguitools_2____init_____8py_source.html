23 """Modules that define the workbench GuiCommands to perform graphical actions.
24 
25 These GUI Commands or tools are called by buttons, menus, contextual menus,
26 toolbars, and other graphical widgets.
27 They are normally loaded by the workbench's `InitGui` module.
28 In this workbench these GUI tools are loaded by the `DraftTools`
29 module, which itself is loaded by `InitGui.DraftWorkbench.Initialize`.
30 
31 This means that any script or workbench that wishes to use
32 this workbench's tools can import the individual modules,
33 or just `DraftTools`, to have access to them.
34 
35 ::
36 
37  import draftguitools.gui_rectangles
38  import draftguitools.gui_circles
39  import draftguitools.gui_arcs
40  import DraftTools
41 
42 These modules should not be imported in a console-only session,
43 as they are not intended to be used without the graphical interface (GUI).
44 They are also not generally suitable for scripting
45 as they normally require graphical input in the task panel or a selection
46 in the 3D view (`Gui.Selection`).
47 
48 Most of these GUI tools require certain components of Draft to exist
49 like the `gui_snapper.Snapper`and the `DraftGui.DraftToolBar` classes.
50 These classes are normally installed in the global `Gui` namespace,
51 so they are accessible at all times.
52 
53 ::
54 
55  import DraftGui
56  import draftguitools.gui_snapper
57  Gui.draftToolBar = DraftGui.DraftToolBar()
58  Gui.Snapper = draftguitools.gui_snapper.Snapper()
59 
60 These classes can be imported and initialized individually
61 but it is easier to set them up just by importing `DraftTools`.
62 """
