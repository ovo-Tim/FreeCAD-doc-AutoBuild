1265 
1266  Determines if a shape appears to have been extruded from some base face, and
1267  extruded at the normal from that base face. IE: it looks like a cuboid.
1268  https://en.wikipedia.org/wiki/Cuboid#Rectangular_cuboid

1270  If this is the case, returns what appears to be the base face, and the vector
1271  used to make that extrusion.
1272 
1273  The base face is determined based on the sortmethod parameter, which can either
1274  be:
1275 
1276  "area" = Of the faces with the smallest area, the one with the lowest z coordinate.
1277  "z" = The face with the lowest z coordinate.
1278  a 3D vector = the face which center is closest to the given 3D point
1279 
1280  Parameters
1281  ----------
1282  shape: <Part.Shape>
1283  Shape to examine.
1284  sortmethod: {"area", "z"}
1285  Which sorting algorithm to use to determine the base face.
1286 
1287  Returns
1288  -------
1289  Extrusion data: list
1290  Two item list containing the base face, and the vector used to create the
1291  extrusion. In that order.
1292  Failure: None
1293  Returns None if the object does not appear to be an extrusion.
1294  """
1295

1297  return None
1298  if not shape.Solids:
1299  return None
1300  if len(shape.Faces) < 3:
1301  return None
1302  # build faces list with normals
1303  faces = []
1304  import Part
1305  for f in shape.Faces:
1306  try:
1307  faces.append([f,f.normalAt(0,0)])
1308  except Part.OCCError:

1311  pairs = []
1312  for i1, f1 in enumerate(faces):
1313  for i2, f2 in enumerate(faces):

1316  pairs.append([i1,i2])

1322  # check if other normals are all at 90 degrees
