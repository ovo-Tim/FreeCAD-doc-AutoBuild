Runs a command for accessing document attribute or method

Parameters
_typetype of document, Gui or App
_docpointer to a document
_cmdcommand string, streamable

Example:

_FCMD_DOC_CMD(Gui,doc,"getObject('" << objName << "')");

Translates to command (assuming doc's name is 'DocName', and and objName contains value 'ObjName'):

Gui.getDocument('DocName').getObject('ObjName')
