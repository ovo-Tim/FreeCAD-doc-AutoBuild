23 """Provides the object code for the PathTwistedArray object.
24 
25 The copies will be placed along a path like a polyline, spline, or bezier
26 curve, and the copies are twisted around the path by a given rotation
27 parameter.
28 
29 This array was developed in order to build a `twisted bridge` object.
30 
31 See https://forum.freecad.org/viewtopic.php?f=23&t=49617
32 
33 A `twisted bridge` would consist of three parts:
34  1. The ribcage composed of a twisted array generated from a frame
35  and a path.
36  2. The `tunnel` object produced by lofting or sweeping the internal twisted
37  profiles of the ribcage along the path.
38  3. The `walkway` object on which the person can stand; it is generated
39  from the path, and the internal width of the ribcage profile.
40 
41 This module builds only the first element, the twisted ribcage.
42 
43 The tunnel and walkway are built with the `twisted bridge`
44 object in the Arch Workbench.
45 """
