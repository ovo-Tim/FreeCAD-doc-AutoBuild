◆ allowedSelSequences

std::vector<std::vector<SketcherGui::SelType> > CmdSketcherConstraint::allowedSelSequences
protected

allowedSelSequences Each element is a vector representing sequence of selections allowable. DrawSketchHandlerGenConstraint will use these to filter elements and generate sequences to be passed to applyConstraint(). Whenever any sequence is completed, applyConstraint() is called, so it's best to keep them prefix-free. Be mindful that when SelVertex and SelRoot are given preference over SelVertexOrRoot, and similar for edges/axes. Thus if a vertex is selected when SelVertex and SelVertexOrRoot are both applicable, only sequences with SelVertex will be continue.
