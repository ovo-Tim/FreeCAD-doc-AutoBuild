Detailed Description

Helper class to return the generated and modified shape given an input shape

Shape history information is extracted using OCCT APIs BRepBuilderAPI_MakeShape::Generated/Modified(). However, there is often some glitches in various derived class. So we use this class as an abstraction, and create various derived classes to deal with the glitches.
