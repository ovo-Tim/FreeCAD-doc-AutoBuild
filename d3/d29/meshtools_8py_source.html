216  """Now we are looking for nodes inside of the Faces = filling the bit_pattern_dict
217  {eleID : [lenEleNodes, binary_position]}
218  see forum post for a very good explanation of what"s really happening
219  https://forum.freecad.org/viewtopic.php?f=18&t=17318&start=50#p141108
220  The bit_pattern_dict holds later an integer (bit array) for each element, which gives us
221  the information we are searching for:
222  Is this element part of the node list (searching for elements)
223  or has this element a face we are searching for?
224  The number in the ele_dict is organized as a bit array.
225  The corresponding bit is set, if the node of the node_set is contained in the element.
226  """
227  FreeCAD.Console.PrintLog("len femnodes_ele_table: " + str(len(femnodes_ele_table)) + "\n")

233  len_ele = len(femelement_table[ele])
234  bit_pattern_dict[ele] = [len_ele, 0]
