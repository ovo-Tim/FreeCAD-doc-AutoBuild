◆ checkAutoHide()

bool OverlayTabWidget::checkAutoHide ( ) const

Check if needs to auto hide this tab widget

Besides when auto hide mode is activated by user, the tab widget will also auto hide if the current view does not have panning capability (queried through MdiView::hasMsg("CanPan")). The user can explicitly disable this auto hide if no pan by setting user parameter View/DockOverlayAutoView, or preference option Display -> UI -> Auto hide in non 3D view.

bool OverlayTabWidget::onEscape ( )

Handle ESC key press

If the this overlay tab widget is hidden and its hint/tabs are visible, pressing ESC will hide the hint and tabs.

If this overlay tab widget is visible and the current mouse cursor is on top of the tab widget title bar or any of its split handler, then pressing ESC will hide the title bar and split handler.
