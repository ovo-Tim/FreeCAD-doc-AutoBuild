Detailed Description

The TextOutputStream class provides writing of ASCII data to an ostream, with custom handling for std::string to make it easier to write a single multi-line (or multi-word) string. This is designed for easy compatibility with the LinkStage3 implementation of the OutputStream class, used to store StringHashers for the toponaming mitigation technique.
