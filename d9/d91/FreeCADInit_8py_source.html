342 class FCADLogger(object):
343  '''Convenient class for tagged logging.
344 
345  Example usage:
346  >>> logger = FreeCAD.Logger('MyModule')
347  >>> logger.info('log test {}',1)
348  24.36053 <MyModule> <input>(1): test log 1
349 
350  The default output format is:
351  <timestamp> <tag> <source file>(line number): message
352 
353  The message is formatted using new style Python string formatting, e.g.
354  'test {}'.format(1). It is strongly recommended to not directly use
355  Python string formatting, but pass additional argument indirectly through
356  various logger print function, because the logger can skip string
357  evaluation in case the logging level is disabled. For more options,
358  please consult the docstring of __init__(), catch() and report().
359 
360  To set/get logger level:
361  >>> FreeCAD.setLogLevel('MyModule','Trace')
362  >>> FreeCAD.getLogLevel('MyModule')
363  4
364 
365  There are five predefined logger level, each corresponding to an integer
366  value, as shown below together with the corresponding logger print
367  method,
368  0: Error, Logger.error()
369  1: Warning, Logger.warn()
370  2: Message, Logger.msg() or info()
371  3: Log, Logger.log() or debug()
372  4: Trace, Logger.trace()
373 
374  FreeCAD.setLogLevel() supports both text and integer value, which allows
375  you to define your own levels. The level set is persisted to user
376  configuration file.
377 
378  By default any tag has a log level of 2 for release, and 3 for debug
379  build.
380  '''
381 
382  _levels = { 'Error':0, 'error':0,
383  'Warning':1, 'warn':1,
384  'Message':2, 'msg':2, 'info':2,
385  'Log':3, 'log':3, 'debug':3,
386  'Trace':4, 'trace':4,}
387  _printer = [
388  FreeCAD.Console.PrintError,
389  FreeCAD.Console.PrintWarning,
390  FreeCAD.Console.PrintMessage,

395  '''Construct a logger instance.
396 
397  Supported arguments are their default values are,
398 
399  * tag: a string tag for this logger. The log level of this logger can be
400  accessed using FreeCAD.getLogLevel(tag)/setLogLevel(tag,level).
401  All logger instance with the same tag shares the same level
402  setting.

404  * printTag (True): whether to print tag
