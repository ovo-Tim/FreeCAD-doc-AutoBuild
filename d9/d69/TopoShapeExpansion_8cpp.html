Encode and set an element name in the elementMap. If a hasher is defined, apply it to the name.

Parameters
elementThe element name(type) that provides 1 one character suffix to the name IF <conditions>.
namesThe subnames to build the name from. If empty, return the TopoShape MappedName.
markerThe elementMap name or suffix to start the name with. If null, use the elementMapPrefix.
opThe op text passed to the element name encoder along with the TopoShape Tag
_sidsIf defined, records the sub ids processed.
Returns
The encoded, possibly hashed name.
