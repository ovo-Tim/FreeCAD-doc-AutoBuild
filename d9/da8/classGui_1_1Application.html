bool Application::activateWorkbench ( const char *  name)

Activate a named workbench.

Activate the matching workbench to the registered workbench handler with name name. The handler must be an instance of a class written in Python. Normally, if a handler gets activated a workbench with the same name gets created unless it already exists.

The old workbench gets deactivated before. If the workbench to the handler is already active or if the switch fails false is returned.
