24 class SceneDetail(object):
25  """SceneDetail class: abstract class for tempovis scene save/restore plug-in. An implementation must provide:
26  * data storage (as "data" attribute of the object)
27  * constructor (preferably, with value for stored data as optional argument)
28  * methods to apply values to actual scene (apply_data),
29  * ...and to read out the state of the detail in the actual scene (scene_value)
30  * keying, for identifying two detail instances that affect the exact same thing
31  * class_id string, which is required for keying
32  * copying
33  * test for equality, that checks if .data attributes of two SceneDetail instances are equal
34  * info on if the modification affects what is saved to disk, and thus should be undone temporarily for file writing.
35  """
