Standard construction from a heap-allocated StringID. This reference-counting class manages the lifetime of the StringID, ensuring it is deallocated when its reference count goes to zero.

Parameters
stringIDA pointer to a StringID allocated with "new"
index(optional) An index value to store along with the StringID. Defaults to zero.
