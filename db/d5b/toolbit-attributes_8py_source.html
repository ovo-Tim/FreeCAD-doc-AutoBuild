24 # Script for manipulating toolbit shapes in a batch. This is handy for making
25 # attributes consistent over multiple or all shape file.
26 #
27 # Most commands are straight forward, except for --set which can be used to
28 # set the actual value of a property, or, in the case of enumerations it can
29 # also be used to set the enum values. This might be particularly useful when
30 # adding more materials.
31 #
32 # The following example moves all properties from the "Extra" group into the
33 # "Attributes" group. Note that the Attributes group might or might not
34 # already exist. If it does exist the specified group gets merged in.
35 #
36 # ./toolbit-attributes.py --move 'Extra:Attributes' src/Mod/CAM/Tools/Shape/*.fcstd
37 #
38 # This example sets the Flutes value of all Shapes to 0:
39 #
40 # ./toolbit-attributes.py --set Flutes=0 src/Mod/CAM/Tools/Shape/*.fcstd
41 #
42 # Finally, this example sets the enumerations of the Material attribute:
43 #
44 # ./toolbit-attributes.py --set 'Material=[HSS,Carbide,Tool Steel,Titanium]' src/Mod/CAM/Tools/Shape/*.fcstd
45 #
46 # After running this tool it might be necessary to open the shape files
47 # manually and make sure they are visible and the thumbprint image is
48 # saved.
49 #
50 # !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
51 # !!! Final note: this is a dangerous tool and should not be shipped with FC. !!!!
52 # !!! It's sole purpose is to make life of the developers easier and ensure !!!!
53 # !!! consistent attributes across all toobit shapes. !!!!
54 # !!! A single typo can ruin a lot of toolbit shapes - make sure to only use !!!!
55 # !!! it if those shape files are under a version control system and you can !!!!
56 # !!! back out the changes easily. !!!!
57 # !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
