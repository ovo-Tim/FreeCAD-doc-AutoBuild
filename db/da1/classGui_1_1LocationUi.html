Detailed Description

template<class Ui>
class Gui::LocationUi< Ui >

This template class does basically the same as LocationDialogUi unless that it doesn inherit from a widget but only from the UI_-generated class. Thus, this class can be used as composition in dialog-based classes without including the ui_-generated header file. The Ui_-class can simply be forward declared, then.
