46 class TaskPanelPolarArray:
47  """TaskPanel code for the PolarArray command.
48 
49  The names of the widgets are defined in the `.ui` file.
50  This `.ui` file `must` be loaded into an attribute
51  called `self.form` so that it is loaded into the task panel correctly.
52 
53  In this class all widgets are automatically created

55 
56  The `.ui` file may use special FreeCAD widgets such as
57  `Gui::InputField` (based on `QLineEdit`) and
