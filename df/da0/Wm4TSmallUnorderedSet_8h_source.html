23 // This template class is for unordered sets of objects. The intent is that
24 // the sets are not too large. If you do not plan on searching the set and
25 // you know that the elements to be added are unique, then the size of the
26 // set is irrelevant since a member function is provided that inserts an
27 // element without checking to see if one already exists.
28 //
29 // The class T is either native data or is class data that has the following
30 // member functions:
31 // T::T ()
32 // T::T (const T&);
33 // T& T::operator= (const T&)
