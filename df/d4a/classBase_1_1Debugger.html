Detailed Description

This is a utility class to break the application at a point to inspect e.g. the result of an algorithm. You usually use it like this

...
Base::Debugger dbg;
dbg.attach();
dbg.exec();
...

Or you can connect it with a button and let the user click it in order to continue.

QPushButton* btn = new QPushButton();
btn->setText("Continue");
btn->show();
Base::Debugger dbg;
connect(btn, &QPushButton::clicked, &dbg, &Debugger::quit);
dbg.exec();
