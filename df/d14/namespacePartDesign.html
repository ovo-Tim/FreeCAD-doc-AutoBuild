static void PartDesign::computeIntersection ( gp_Pnt  pa1,
gp_Pnt  pa2,
gp_Pnt  pb1,
gp_Pnt  pb2,
double &  x,
double &  y 
)
static

Computes 2D intersection between the lines (pa1, pa2) and (pb1, pb2). The lines are assumed to be crossing, and it is an error to specify parallel lines. Only the x and y coordinates of the points are used to compute the 2D intersection.

The result are the x and y coordinate of the intersection point.
