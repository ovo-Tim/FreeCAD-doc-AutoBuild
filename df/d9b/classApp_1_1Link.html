Extends the rules for handling property name changed, so that extensions are given an opportunity to handle it. If an extension handles a change, neither the rest of the extensions, nor the container itself get to handle it.

Extensions get their extensionHandleChangedPropertyName() called.

If no extension handles the request, then the containers handleChangedPropertyName() is called.
