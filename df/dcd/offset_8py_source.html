46  """offset(object,delta,[copymode],[bind])
47 
48  Offset the given wire by applying the given delta Vector to its first
49  vertex.
50 
51  Parameters
52  ----------
53  obj :
54 
55  delta : Base.Vector or list of Base.Vector
56  If offsetting a BSpline, the delta must not be a Vector but a list
57  of Vectors, one for each node of the spline.
58 
59  copy : bool
60  If copymode is True, another object is created, otherwise the same
61  object gets offsetted.
62 
63  copy : bool
64  If bind is True, and provided the wire is open, the original
65  and the offset wires will be bound by their endpoints, forming a face.
66 
67  sym : bool
68  if sym is True, bind must be true too, and the offset is made on both
69  sides, the total width being the given delta length.
