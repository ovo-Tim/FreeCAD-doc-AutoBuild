Detailed Description

This class calculates a B-spline area on any point cloud (AKA scattered data). The surface is generated iteratively with the help of a parameter correction. See Hoschek/Lasser 2nd ed. (1992). The approximation is expanded to include smoothing terms so that smooth surfaces can be generated.
