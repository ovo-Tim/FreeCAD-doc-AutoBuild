30  // The segment is represented as P+t*D, where P is the segment origin,
31  // D is a unit-length direction vector and |t| <= e. The value e is
32  // referred to as the extent of the segment. The end points of the
33  // segment are P-e*D and P+e*D. The user must ensure that the direction
34  // vector is unit-length. The representation for a segment is analogous
35  // to that for an oriented bounding box. P is the center, D is the
36  // axis direction, and e is the extent.
