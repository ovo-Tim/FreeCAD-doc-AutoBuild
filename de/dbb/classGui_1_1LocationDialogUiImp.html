Detailed Description

This is a subclass of LocationDialog using AbstractUi that implements the pure virtual methods of its base class. Other dialog-based classes can directly inherit from this class if the location-interface is required. The advantage of this class compared to LocationDialogImp is that the ui_-header file doesn't need to be included in the header file of its sub-classes because it uses "type erasure with templates".
