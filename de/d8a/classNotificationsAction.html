Detailed Description

Drop menu Action containing the notifications widget. It stores all the notification item information in the form of NotificationItems (QTreeWidgetItem). This information is used by the Widget and by the non-intrusive messages. It owns the notification resources and is responsible for the release of the memory resources, either directly for the intermediate fast cache or indirectly via QT for the case of the QTreeWidgetItems.
