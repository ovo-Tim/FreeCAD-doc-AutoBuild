Detailed Description

The MeshEvaluation class checks the mesh kernel for correctness with respect to a certain criterion, such as manifoldness, self-intersections, etc. The passed mesh kernel is read-only and cannot be modified.

See also
MeshEvalTopology
MeshEvalGeometry The class itself is abstract, hence the method Evaluate() must be implemented by subclasses.

virtual bool MeshCore::MeshEvaluation::Evaluate ( )
pure virtual

Evaluates the mesh kernel with respect to certain criteria. Must be reimplemented by every subclass. This pure virtual function returns false if the mesh kernel is invalid according to this criterion and true if the mesh kernel is correct.
