Detailed Description

3D Annotation Node - Annotation with depth buffer

This class is just like SoAnnotation with the difference that it does not disable the depth buffer instead it clears it and renders on top of everything with proper depth control.

It should be used with caution as it does clear the depth buffer for each annotation!
