Detailed Description

The Logger Interface This class describes an Interface for logging within FreeCAD. If you want to add a new "sink" to FreeCAD's logging mechanism, then inherit this class. You'll also need to register your derived class with ConsoleSingleton.

pure virtual

Used to send a Log message at the given level.

Parameters
notifiernameA string identifying the entity generating the notification. It may be the Label of the App::Document or the full label of the App::DocumentObject.
msgThe message to be notified.
levelA valid level (Log, Message, Error, Notification, CriticalNotification, ...)
recipientA valid intended recipient (All, Developer, User). Observers may decide to process only notifications when a certain recipient is intended.
contentA valid content property (Untranslated, Translatable, Untranslatable). Observers may decide not to process notifications if they are not translated or cannot be translated (are untranslatable). Or conversely, may decide not to process already translated notifications. It is up to the intended behaviour of the observer.
