147 # 15.3 ATan - arithmetic function
148 # FUNCTION ATAN ( V1:NUMBER; V2:NUMBER ) : REAL;
149 # The atan function returns the angle given a tangent value of V , where V is given by the
150 # expression V = V1/V2.
151 # Parameters :
152 # a) V1 is a number.
153 # b) V2 is a number.
154 # Result : The angle in radians (-pi/2<=result<=pi/2) whose tangent is V. If V2 is zero, the result
155 # is pi/2 or -pi/2 depending on the sign of V1.
156 # Conditions : Both V1 and V2 shall not be zero.
157 # EXAMPLE 128 { ATAN ( -5.5, 3.0 ) --> -1.071449...
158 def ATAN(V1, V2):
159  if not isinstance(V1, NUMBER) and not isinstance(V2, NUMBER):
160  raise TypeError("ATAN function takes 2 NUMBER parameters")
161  if V2 == 0:
162  if V1 > 0:
163  return REAL(math.pi / 2)
164  elif V1 < 0:
165  return REAL(-math.pi / 2)
166  else:
167  raise ValueError("ATAN parameters can be both equal to zero")
168  else:
169  return REAL(math.atan(float(V1) / float(V2)))

253 # 15.9 Format - general function
254 # FUNCTION FORMAT(N:NUMBER; F:STRING):STRING;
255 # The format returns a formatted string representation of a number.
256 # Parameters :
257 # a) N is a number (integer or real).
258 # b) F is a string containing formatting commands.
259 # Result : A string representation of N formatted according to F. Rounding is applied to the
260 # string representation if necessary.
261 # The formatting string contains special characters to indicate the appearance of the result. The
262 # formatting string can be written in three ways:
263 # a) The formatting string can give a symbolic description of the output representation.
264 # b) The formatting string can give a picture description of the output representation.
265 # c) When the formatting string is empty, a standard output representation is produced.
266 # Table 20:
267 # Number Format Display Comment
268 # 10 +7I ' +10' Zero suppression
269 # 10 +07I '+000010' Zeros not suppressed
270 # 10 10.3E ' 1.000E+01'
271 # 123.456789 8.2F ' 123.46'
272 # 123.456789 8.2E '1.23E+02'
273 # 123.456789 08.2E '0.12E+02' Preceding zero forced
274 # 9.876E123 8.2E '9.88E+123' Exponent part is 3 characters
275 # and width ignored
276 # 32.777 6I ' 33' Rounded
277 # Python definition
278 # =================
279 # python string formatting is obtained from the val function
280 # @TODO: implement a safe eval or provide another implementation
281 # that avoids unsafe eval python builtin function.
282 def FORMAT(N, F):
283  if not isinstance(N, NUMBER):
284  raise TypeError("FORMAT function takes a NUMBER parameter")
285  if not isinstance(F, STRING):
286  raise TypeError("FORMAT function takes a NUMBER parameter")
287  py_formatting = F.lower()
288  string_to_evaluate = "'%"
289  string_to_evaluate += "%s'" % py_formatting
290  string_to_evaluate += "%"
291  string_to_evaluate += "%s" % N
292  result = eval(string_to_evaluate).upper()
293  return STRING(result)
