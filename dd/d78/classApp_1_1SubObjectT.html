◆ NormalizeOption

enum App::SubObjectT::NormalizeOption : uint8_t
strong

Options used by normalize()

Enumerator
NoElement 

Do not include sub-element reference in the output path.

NoFlatten 

Do not flatten the output path. If not specified, the output path will be flatten to exclude intermediate objects that belong to the same geo feature group before resolving. For example, Part.Fusion.Box. -> Part.Box.

KeepSubName 

Do not change the sub-object component inside the path. Each component of the subname object path can be either the object internal name, the label of the object if starts with '$', or an integer index. If this option is not specified, each component will be converted to object internal name, except for integer index.

ConvertIndex 

Convert integer index in the path to sub-object internal name
