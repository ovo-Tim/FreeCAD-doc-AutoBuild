◆ beginCharStream()

std::ostream & Writer::beginCharStream ( CharStreamFormat  format = CharStreamFormat::Raw)

Create an output stream for storing character content The input is assumed to be valid character with the current XML encoding, and will be enclosed inside CDATA section. The stream will scan the input and properly escape any CDATA ending inside.

Parameters
formatIf Base64Encoded, the input will be base64 encoded before storing. If Raw, the input is assumed to be valid character with the current XML encoding, and will be enclosed inside CDATA section. The stream will scan the input and properly escape any CDATA ending inside.
Returns
Returns an output stream.

You must call endCharStream() to end the current character stream.
