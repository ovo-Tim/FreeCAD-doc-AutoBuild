166 # The joint object consists of 2 JCS (joint coordinate systems) and a Joint Type.
167 # A JCS is a placement that is computed (unless it is detached) from references (PropertyXLinkSubHidden) that links to :
168 # - An object: this can be any Part::Feature solid. Or a PartDesign Body. Or a App::Link to those.
169 # - An element name: This can be either a face, an edge, a vertex or empty. Empty means that the Object placement will be used
170 # - A vertex name: For faces and edges, we need to specify which vertex of said face/edge to use
171 # Both element names hold the full path to the object.
172 # From these a placement is computed. It is relative to the Object.

639  """
640  So here we want to find a placement that corresponds to a local coordinate system that would be placed at the selected vertex.
641  - obj is usually a App::Link to a PartDesign::Body, or primitive, fasteners. But can also be directly the object.1
642  - elt can be a face, an edge or a vertex.
643  - If elt is a vertex, then vtx = elt And placement is vtx coordinates without rotation.
644  - if elt is an edge, then vtx = edge start/end vertex depending on which is closer. If elt is an arc or circle, vtx can also be the center. The rotation is the plane normal to the line positioned at vtx. Or for arcs/circle, the plane of the arc.
645  - if elt is a plane face, vtx is the face vertex (to the list of vertex we need to add arc/circle centers) the closer to the mouse. The placement is the plane rotation positioned at vtx
646  - if elt is a cylindrical face, vtx can also be the center of the arcs of the cylindrical face.

690  def preSolve(self, joint, savePlc=True):
691  # The goal of this is to put the part in the correct position to avoid wrong placement by the solve.
692 
693  # we actually don't want to match perfectly the JCS, it is best to match them
694  # in the current closest direction, ie either matched or flipped.
695 
696  sameDir = self.areJcsSameDir(joint)
697  assembly = self.getAssembly(joint)
698 
699  part1 = UtilsAssembly.getMovingPart(assembly, joint.Reference1)
700  part2 = UtilsAssembly.getMovingPart(assembly, joint.Reference2)
701 
702  isAssembly = assembly.Type == "Assembly"
703  if isAssembly:
704  joint.Activated = False
705  part1Connected = assembly.isPartConnected(part1)
706  part2Connected = assembly.isPartConnected(part2)
707  joint.Activated = True
708  else:
709  part1Connected = True
710  part2Connected = False
711 
712  if not part1Connected:
713  if savePlc:
714  self.partMovedByPresolved = part1
715  self.presolveBackupPlc = part1.Placement
716 
717  globalJcsPlc2 = UtilsAssembly.getJcsGlobalPlc(joint.Placement2, joint.Reference2)
718  jcsPlc1 = UtilsAssembly.getJcsPlcRelativeToPart(
719  assembly, joint.Placement1, joint.Reference1
720  )
721  if not sameDir:
722  jcsPlc1 = UtilsAssembly.flipPlacement(jcsPlc1)
723
