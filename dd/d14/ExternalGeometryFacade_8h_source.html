37 // This class is a Facade to handle EXTERNAL sketcher geometry and sketcher geometry extensions with
38 // a single sketcher specific interface.
39 //
40 // Exactly the same considerations as for GeometryFacade apply (see documentation of
41 // GeometryFacade).
42 //
43 // It was not made publicly deriving from GeometryFacade because it is not possible to differentiate
44 // functions by return type, which is the case of getFacade() returning a unique_ptr to
45 // GeometryFacade in GeometryFacade, and one to ExternalGeometryFacade. I have not managed to find a
46 // good solution to this problem, thus the code duplication.
47 //
48 // Summary Remarks:
49 // It is intended to have a separate type (not being a Geometry type).
50 // it is intended to have the relevant interface in full for the sketcher extension only
51 // It is intended to work on borrowed memory allocation.
